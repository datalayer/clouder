# Usage

```bash
# Help.
clouder --help
clouder --help-all
```

```bash
# Context.
clouder context -h
clouder context ls
clouder context set aws <ovh_project_id>
clouder context set ovh <aws_account_id>
```

```bash
# Box.
clouder box ls
clouder box info
```

```bash
# Kubernetes.
clouder k8s ls
```

```bash
# SSH Key.
clouder ssh-key ls
```
