# Usage

```bash
# Infos.
clouder about
clouder --help
clouder --help-all
```

```bash
# Context.
clouder ctx -h
clouder ctx ls
clouder ctx set aws `ovh_project_id`
clouder ctx set ovh `aws_account_id`
```

```bash
# Box.
clouder box ls
clouder box info
```

```bash
# Kubernetes.
clouder k8s create k1
clouder k8s kubeconfig k1
clouder k8s create-nodepool k1 jupyter-small --flavor b2-7 --min 1 --desired 1 --max 10 --role jupyter --variant small --xpu cpu
clouder k8s
```

```bash
# SSH Key.
clouder ssh-key ls
```

```bash
# Virtual Machines.
clouder vm ls
```

## Operator

```bash
# Start the Clouder operator.
clouder operator start
```

```bash
# Stop the Clouder operator.
clouder operator stop
```

## Server

```bash
# Start the Clouder server.
clouder server
```
