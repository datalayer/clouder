# Usage

```bash
# Infos.
clouder about
clouder --help
clouder --help-all
```

```bash
# Context.
clouder ctx -h
clouder ctx ls
clouder ctx set aws `ovh_project_id`
clouder ctx set ovh `aws_account_id`
```

```bash
# Box.
clouder box ls
clouder box info
```

```bash
# Kubernetes.
clouder k8s create k1
clouder k8s kubeconfig k1
clouder k8s create-nodepool k1 nodepool_jupyter b2-7 1 1 10 jupyter cpu
clouder k8s ls
```

```bash
# SSH Key.
clouder ssh-key ls
```

```bash
# Virtual Machines.
clouder vm list
```

## Operator

```bash
# Start the Clouder operator.
clouder operator start
```

```bash
# Stop the Clouder operator.
clouder operator stop
```

## Server

```bash
# Start the Clouder server.
clouder server
```
